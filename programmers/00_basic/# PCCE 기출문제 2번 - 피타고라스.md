# PCCE 2번 - 피타고라스의 정리

## 📌 문제 설명
직각삼각형에서 빗변의 제곱은 다른 두 변의 제곱의 합과 같습니다.

공식:
c² = a² + b²

정수 a(한 변)과 c(빗변)가 주어질 때,
다른 한 변의 제곱값 b²을 구하는 문제입니다.

공식 변형:
b² = c² - a²

---

## 🧠 풀이 접근 방식
1. 직각삼각형 공식을 활용
2. b² = c² - a²로 수학적 변형
3. 정수 연산을 통해 계산 결과 출력

---

## ⏱ 시간 복잡도
O(1)

---

## 🎯 구현 의도
수학 공식을 단순 적용하는 것이 아니라,
공식을 문제 조건에 맞게 변형하여 코드로 구현하는 연습을 목표로 하였습니다.
