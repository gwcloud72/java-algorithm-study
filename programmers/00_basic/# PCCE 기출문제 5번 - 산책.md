# PCCE 5 - 산책

🔗 문제 출처: Programmers PCCE 기출문제 5번

---

## 📌 문제 설명

동/서/남/북 방향으로 1m씩 이동하는 경로가 문자열로 주어진다.  
이동을 모두 수행한 뒤 최종 좌표를 `[동쪽 거리, 북쪽 거리]` 형태로 반환한다.

### 이동 규칙

- N → 북쪽 +1
- S → 북쪽 -1
- E → 동쪽 +1
- W → 동쪽 -1

---

## 🧠 풀이 아이디어

- 현재 위치를 나타내는 두 변수 사용
  - `east` : 동/서 방향 좌표
  - `north` : 남/북 방향 좌표
- 문자열을 한 글자씩 순회
- 각 문자에 따라 좌표를 증감

---

## 시간 복잡도

문자열을 한 번 순회 → O(N)
