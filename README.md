# ✏️ Java Algorithm Study

코딩 테스트 준비하면서  
Java로 푼 알고리즘 문제를 정리하는 저장소입니다.

문제를 풀고 끝내는 것보다  
어떻게 이해했고 어떤 식으로 풀었는지 같이 정리하려고 만들었습니다.  
나중에 다시 봤을 때도 바로 이해할 수 있게 기록하는 걸 목표로 하고 있습니다.

---

## 🎯 공부 목표

- 자료구조와 알고리즘 기본기 다시 잡기
- 여러 문제를 풀면서 문제 풀이 감각 익히기
- 시간 복잡도 계산에 익숙해지기
- 보기 편한 코드로 작성하는 연습하기

---

## 📁 저장소 구조

문제는 사이트별로 나눠서 정리하고 있습니다.

```bash
├── programmers/
├── baekjoon/
```

문제가 더 쌓이면  
난이도나 유형별로 폴더를 조금 더 나눌 예정입니다.

---

## 🛠 풀이 정리 방식

문제를 풀 때는 보통 아래 순서로 정리합니다.

1. 문제 내용 확인
2. 풀이 방법 생각하기
3. Java로 구현하기
4. 다시 보면서 부족한 부분 정리하기
