## 📌 커밋 메시지

문제 풀이 기록을 보기 쉽게 하기 위해 아래 형식으로 커밋합니다.

```
[사이트] 문제번호 - 문제이름
```

예시:

```
[Programmers] 120808 - 분수의 덧셈
[Baekjoon] 10818 - 최소, 최대
[LeetCode] 1 - Two Sum
```

가능하면 한 문제당 하나의 커밋으로 관리합니다.
