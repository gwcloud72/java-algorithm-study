# 10807번 - 개수 세기

🔗 문제 출처: BOJ 10807

---

## 📌 문제 설명

총 N개의 정수가 주어진다.  
이 중에서 정수 v가 몇 개인지 출력하는 문제.

- 1 ≤ N ≤ 100
- 정수 범위: -100 ≤ 값 ≤ 100

---

## 🧠 풀이 아이디어

- 정수 N을 입력받는다.
- N개의 정수를 입력받는다.
- 찾고 싶은 정수 v를 입력받는다.
- N개의 정수를 하나씩 확인하면서
  - 현재 값이 v와 같으면 count 증가
- 최종 count 출력

---

## ⏱ 시간 복잡도

- 배열을 한 번 순회 → O(N)

---

## 💡 배운 점

- 반복문을 이용한 개수 세기
- 조건문을 활용한 비교
- BufferedReader와 StringTokenizer 사용
