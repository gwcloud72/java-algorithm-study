# 4344번 - 평균은 넘겠지

🔗 문제 출처: BOJ 4344

---

## 📌 문제 설명

각 테스트 케이스마다:

1. 학생 수 N이 주어진다.
2. N명의 점수가 주어진다.
3. 평균을 넘는 학생의 비율을 구해서 출력한다.

출력은 소수점 셋째 자리까지 표시하고 `%`를 붙인다.

---

## 🧠 풀이 아이디어

- 먼저 테스트 케이스 수 T를 입력받는다.
- 각 테스트 케이스마다:
  - 학생 수 N과 점수들을 입력받는다.
  - 총합을 구해 평균을 계산한다.
  - 평균보다 높은 학생 수를 센다.
  - (평균 초과 학생 수 / 전체 학생 수) * 100을 계산한다.
  - 소수점 셋째 자리까지 출력한다.

---

## ⏱ 시간 복잡도

- 각 테스트 케이스마다 N번 순회 → O(N)

---

## 💡 배운 점

- 평균 계산
- 조건 비교
- 비율 계산
- printf 포맷팅 (%.3f%%)
