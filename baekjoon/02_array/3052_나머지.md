# 3052번 - 나머지

🔗 문제 출처: BOJ 3052

---

## 📌 문제 설명

정수 10개를 입력받는다.  
각 수를 42로 나눈 나머지를 구한 뒤, 서로 다른 나머지의 개수를 출력한다.

---

## 🧠 풀이 아이디어

- 나머지는 0 ~ 41 범위이므로 길이 42의 boolean 배열을 사용한다.
- 입력을 10번 받으면서 `x % 42` 값을 계산한다.
- 해당 나머지를 `visited[remainder] = true`로 체크한다.
- 마지막에 `true`의 개수를 세어 출력한다.

---

## ⏱ 시간 복잡도

- 입력 10개 처리 및 배열 순회 → O(1) (일반화하면 O(N))

---

## 💡 배운 점

- 나머지 연산 `%`
- boolean 배열로 중복 체크
- 체크된 값 개수 세기
