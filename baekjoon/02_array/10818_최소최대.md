# 10818번 - 최소, 최대

🔗 문제 출처: BOJ 10818

---

## 📌 문제 설명

정수 N개가 주어진다.  
이 중에서 최솟값과 최댓값을 출력한다.

---

## 🧠 풀이 아이디어

1. 정수 N을 입력받는다.
2. N개의 정수를 한 줄로 입력받는다.
3. 첫 번째 값을 기준으로 min과 max를 초기화한다.
4. 나머지 값들과 비교하면서
   - 더 작으면 min 갱신
   - 더 크면 max 갱신
5. 최종 min과 max를 출력한다.

---

## ⏱ 시간 복잡도

- 배열을 한 번 순회 → O(N)

---

## 💡 배운 점

- 최소/최대 초기화 방법
- 반복문을 이용한 비교
- StringTokenizer로 여러 값 입력 처리
