# 10871번 - X보다 작은 수

🔗 문제 출처: BOJ 10871

## 📌 문제 설명
정수 N개가 주어졌을 때, X보다 작은 수를 모두 출력한다.

## 🧠 풀이 아이디어
- 첫 줄에서 N과 X를 입력받는다.
- 둘째 줄에서 N개의 정수를 입력받는다.
- 반복문으로 각 숫자를 확인한다.
- X보다 작은 수만 StringBuilder에 추가한다.
- 마지막에 결과를 출력한다.

## ⏱ 시간 복잡도
배열을 한 번 순회 → O(N)

## 💡 배운 점
- 조건에 맞는 값만 출력
- StringTokenizer 사용
- StringBuilder 활용
