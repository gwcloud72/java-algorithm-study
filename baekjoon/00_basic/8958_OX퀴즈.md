# 8958번 - OX퀴즈

🔗 문제 출처: BOJ 8958

## 📌 문제 설명
OX 퀴즈 결과 문자열이 주어진다.

- O를 맞히면 점수를 얻는다.
- 연속된 O는 점수가 1점씩 증가한다.
- X가 나오면 연속 점수는 0으로 초기화된다.

각 테스트 케이스마다 총점을 출력한다.

## 🧠 풀이 아이디어
- 테스트 케이스 수 T를 입력받는다.
- 각 문자열을 한 글자씩 확인한다.
- O이면 연속 점수 증가 후 total에 더한다.
- X이면 연속 점수를 0으로 초기화한다.
- 각 테스트 케이스의 total을 출력한다.

## ⏱ 시간 복잡도
문자열을 한 번 순회 → O(N)

## 💡 배운 점
- 문자열 순회
- 연속된 값 누적 처리
- 조건문 활용
