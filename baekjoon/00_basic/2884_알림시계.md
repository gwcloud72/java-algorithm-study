# 2884번 - 알람 시계

🔗 문제 출처: BOJ 2884

## 📌 문제 설명
현재 설정된 시각이 주어졌을 때, 알람을 45분 앞선 시간으로 바꾸는 문제이다.

예를 들어:
- 10:10 → 9:25
- 0:30 → 23:45

## 🧠 풀이 아이디어
현재 시각에서 45분을 뺀다.

### 경우 1
분이 45 이상이면:
- 시간은 그대로
- 분만 45 빼기

### 경우 2
분이 45 미만이면:
- 시간을 1 줄여야 함
- 분은 60을 더한 뒤 45 빼기
- 만약 시간이 0이면 23으로 바꾼다

## ⏱ 시간 복잡도
상수 연산 → O(1)

## 💡 배운 점
- 시간 계산
- 조건문 분기
- 0시 처리 예외
