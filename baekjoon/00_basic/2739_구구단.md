# 2739번 - 구구단

🔗 문제 출처: BOJ 2739

---

## 📌 문제 설명

정수 N이 주어졌을 때 N단을 출력하는 프로그램을 작성한다.

출력 형식:

N * 1 = N
N * 2 = 2N
...
N * 9 = 9N

---

## 🧠 풀이 아이디어

- 반복문을 이용해 1부터 9까지 반복한다.
- 각 반복마다 N * i 값을 출력한다.
- 출력 형식에 맞게 문자열을 구성한다.

---

## ⏱ 시간 복잡도

- 1부터 9까지 반복 → O(1)

---

## 💡 배운 점

- for 반복문 사용
- 출력 형식 맞추기
- 문자열과 숫자 결합 출력
