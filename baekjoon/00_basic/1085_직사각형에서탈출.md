# 1085번 - 직사각형에서 탈출

🔗 문제 출처: BOJ 1085

## 📌 문제 설명
점 (x, y)가 직사각형 내부에 있을 때, 직사각형 경계선까지 가는 최소 거리를 구한다.

직사각형의 오른쪽 위 꼭짓점은 (w, h)이다.

## 🧠 풀이 아이디어
점에서 각 변까지의 거리를 구한다.

- 왼쪽까지 거리: x
- 아래까지 거리: y
- 오른쪽까지 거리: w - x
- 위까지 거리: h - y

이 네 값 중 최솟값을 출력한다.

## ⏱ 시간 복잡도
상수 연산 → O(1)

## 💡 배운 점
- 최소값 비교
- 좌표와 거리 개념
- Math.min 사용
