# 2753번 - 윤년

🔗 문제 출처: BOJ 2753

---

## 📌 문제 설명

연도가 주어졌을 때 윤년이면 1, 아니면 0을 출력한다.

### 윤년 조건

- 4로 나누어 떨어지고
- 100으로 나누어 떨어지지 않거나
- 400으로 나누어 떨어지면 윤년

---

## 🧠 풀이 아이디어

윤년 조건을 그대로 코드로 구현한다.

조건식:

(year % 4 == 0 && year % 100 != 0) || year % 400 == 0

해당 조건이 참이면 1, 아니면 0 출력.

---

## ⏱ 시간 복잡도

- 상수 연산 → O(1)

---

## 💡 배운 점

- 조건문 논리 연산자 (&&, ||)
- 나머지 연산 (%)
- 복합 조건 처리
