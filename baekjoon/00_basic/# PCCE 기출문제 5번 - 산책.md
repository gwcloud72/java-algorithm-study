# PCCE 5 - 산책

🔗 문제 출처: Programmers PCCE 기출문제 5번

## 문제 요약

동/서/남/북 방향으로 1m씩 이동하는 경로가 문자열로 주어진다.  
이동을 모두 수행한 뒤 최종 좌표를 `[동쪽 거리, 북쪽 거리]` 형태로 반환하는 문제.

- N → 북쪽 +1
- S → 북쪽 -1
- E → 동쪽 +1
- W → 동쪽 -1

---

## 접근 방법

문자열을 한 번 순회하면서 현재 좌표를 누적 계산했다.

- `east` : 동/서 방향 관리
- `north` : 남/북 방향 관리
- switch 문으로 분기 처리

---

## 빈칸에 채운 내용

- S → `north--`
- E → `east++`
- W → `east--`

---

## 시간 복잡도

문자열을 한 번 순회 → O(N)
