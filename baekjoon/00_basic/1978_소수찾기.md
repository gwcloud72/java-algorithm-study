# 1978번 - 소수 찾기

🔗 문제 출처: BOJ 1978

## 📌 문제 설명
주어진 정수들 중 소수가 몇 개인지 출력한다.

소수는 1보다 크고, 1과 자기 자신만으로 나누어지는 수이다.

## 🧠 풀이 아이디어
- 숫자를 하나씩 확인한다.
- 각 숫자에 대해 소수인지 판별한다.
- 소수이면 count를 증가시킨다.

### 소수 판별 방법
- 1은 소수가 아니다.
- 2부터 n-1까지 나누어 떨어지는 수가 있으면 소수가 아니다.
- 끝까지 없으면 소수이다.

## ⏱ 시간 복잡도
숫자 하나당 O(N) 판별  
전체는 입력 개수에 비례

## 💡 배운 점
- 소수 판별 기본
- 함수 분리
- 조건문과 반복문 활용
