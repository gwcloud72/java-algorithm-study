# 4153번 - 직각삼각형

🔗 문제 출처: BOJ 4153

## 📌 문제 설명
세 변의 길이가 주어질 때, 직각삼각형인지 판별한다.

- 직각삼각형이면 `right`
- 아니면 `wrong`

입력이 `0 0 0`이면 종료한다.

## 🧠 풀이 아이디어
- 세 변 중 가장 큰 값을 찾는다.
- 가장 큰 변을 빗변이라고 생각한다.
- `빗변² == 나머지 두 변²의 합` 이면 직각삼각형이다.
- 반복 입력이므로 while문으로 처리한다.

## ⏱ 시간 복잡도
각 테스트마다 상수 연산 → O(1)

## 💡 배운 점
- 피타고라스 정리 활용
- 최댓값 찾기
- 반복 종료 조건 처리
