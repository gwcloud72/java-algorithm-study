# 2908번 - 상수

🔗 문제 출처: BOJ 2908

---

## 📌 문제 설명

두 개의 세 자리 수가 공백으로 구분되어 주어진다.  
상수는 숫자를 거꾸로 읽기 때문에, 각 수를 뒤집어서 비교한 뒤 더 큰 값을 출력한다.

---

## 🧠 풀이 아이디어

1. 두 숫자를 문자열로 입력받는다.
2. 각 문자열을 뒤집는다.
3. 뒤집은 문자열을 정수로 변환한다.
4. 두 값을 비교해서 더 큰 값을 출력한다.

---

## ⏱ 시간 복잡도

- 문자열 길이가 3이라 사실상 O(1) (일반화하면 O(N))

---

## 💡 배운 점

- 문자열 뒤집기 (StringBuilder의 reverse())
- 문자열 ↔ 정수 변환
- 비교 후 출력
