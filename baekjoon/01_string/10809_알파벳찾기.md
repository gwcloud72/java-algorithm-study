# 10809번 - 알파벳 찾기

🔗 문제 출처: BOJ 10809

---

## 📌 문제 설명

소문자 단어가 주어진다.  
알파벳 a부터 z까지 각각에 대해, 해당 알파벳이 **처음 등장하는 위치(0부터 시작)**를 출력한다.

- 단어에 해당 알파벳이 없다면 `-1`을 출력한다.

---

## 🧠 풀이 아이디어

- 길이 26의 정수 배열 `pos`를 만들어 모든 값을 `-1`로 초기화한다.
- 문자열을 왼쪽부터 한 글자씩 확인한다.
- 현재 문자가 처음 나온 경우에만 해당 위치를 저장한다.

문자 인덱스는 다음처럼 계산한다.
- `idx = ch - 'a'`  (a → 0, b → 1, ..., z → 25)

---

## ⏱ 시간 복잡도

- 문자열을 한 번 순회 → O(N)

---

## 💡 배운 점

- 문자에서 알파벳 인덱스 계산 (`ch - 'a'`)
- 배열 초기화와 조건 처리
- StringBuilder로 출력 최적화
