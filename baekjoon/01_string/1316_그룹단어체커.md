# 1316번 - 그룹 단어 체커

🔗 문제 출처: BOJ 1316

---

## 📌 문제 설명

단어에서 각 문자가 연속해서만 등장하면 그룹 단어이다.

예:
- happy → 그룹 단어
- aba → 그룹 단어 아님 (a가 떨어져서 다시 등장)

N개의 단어가 주어질 때 그룹 단어의 개수를 출력한다.

---

## 🧠 풀이 아이디어

단어를 왼쪽부터 한 글자씩 확인한다.

- 이전 문자(prev)와 현재 문자가 같으면 계속 진행
- 이전 문자와 현재 문자가 다르면(새로운 문자 등장)
  - 그 문자가 이미 나온 적이 있으면 그룹 단어가 아님
  - 처음 나온 문자면 방문 처리

이를 위해 boolean[26] 배열을 사용한다.

---

## ⏱ 시간 복잡도

각 단어를 한 번 순회 → O(L)  
전체는 O(총 문자 수)

---

## 💡 배운 점

- 이전 문자 비교로 “연속” 판별
- boolean 배열로 “이미 등장했는지” 체크
- 문자열 순회 기본기
