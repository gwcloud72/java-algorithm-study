# 2941번 - 크로아티아 알파벳

🔗 문제 출처: BOJ 2941

## 📌 문제 설명
크로아티아 알파벳은 특정 문자열 조합을 하나의 문자로 취급한다.

c= , c-
dz=
d-
lj , nj
s= , z=

문자열이 주어졌을 때 위 조합들을 한 글자로 계산한 총 문자 개수를 출력한다.

## 🧠 풀이 아이디어
- 크로아티아 알파벳 조합을 배열로 저장
- replace로 "*"로 치환
- 최종 문자열 길이 출력
- dz= 는 z=보다 먼저 처리

## ⏱ 시간 복잡도
O(N)

## 💡 배운 점
- 문자열 치환
- 패턴 우선순위
- 문자열 길이 활용
