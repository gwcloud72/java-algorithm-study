# 1157번 - 단어 공부

🔗 문제 출처: BOJ 1157

---

## 📌 문제 설명

알파벳 대소문자가 섞인 단어가 주어진다.

- 가장 많이 사용된 알파벳을 출력한다.
- 대소문자는 구분하지 않는다.
- 가장 많이 사용된 알파벳이 여러 개라면 `?`를 출력한다.

---

## 🧠 풀이 아이디어

1. 입력 문자열을 모두 대문자로 변환한다.
2. 길이 26의 정수 배열을 만들어 알파벳 개수를 센다.
3. 가장 큰 개수를 찾는다.
4. 최댓값이 여러 개인지 확인한다.
5. 하나면 해당 문자 출력, 여러 개면 `?` 출력.

---

## ⏱ 시간 복잡도

- 문자열을 한 번 순회 → O(N)

---

## 💡 배운 점

- 대소문자 변환 (`toUpperCase()`)
- 알파벳 인덱스 계산 (`ch - 'A'`)
- 최댓값 탐색 및 중복 처리
